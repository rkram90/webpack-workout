Welcome to world of Webpack

Webpack is a popular open-source JavaScript module bundler.

It takes the various assets, such as JavaScript, CSS, and images, and bundles them together in a way that is optimized for the web.

Webpack is commonly used in modern web development to manage dependencies, improve performance, and simplify the deployment of web applications.

Webpack can be broken down into these 5 principals:

  • 1. Entry
  • 2. Output
  • 3. Loaders
  • 4. Plugins
  • 5. Mode
